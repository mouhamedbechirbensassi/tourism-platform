\# Mini E-Tourism Platform



A full-stack mini platform for managing hotels and visualizing sales statistics.



\## Tech Stack



Frontend:

\- React 18

\- TypeScript

\- Vite

\- Recharts

\- Hooks only (no Redux)



Backend:

\- Node.js

\- Express

\- TypeScript

\- In-memory data storage



---



\## Features



\- Hotels CRUD (Create / Read / Update / Delete)

\- Dynamic KPI dashboard

\- Monthly revenue line chart

\- Year filter with automatic chart update
